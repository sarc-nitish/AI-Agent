------------------------------
Section 2: Technologies Used
------------------------------
- Python: Core language for building tools and agents.
- Streamlit: Used for building interactive web-based AI dashboards.
- GPT (via OpenAI and Groq): Large Language Models used for natural language understanding and generation.
- LlamaIndex: Used to index and query documents.
- LangChain: To manage LLM chains and retrieval-augmented generation.
- Machine Learning: Concepts such as classification, regression used in backend logic.
- Deep Learning: Used for advanced models such as image recognition or video understanding.

------------------------------
Section 3: AI Use Cases Built
------------------------------
- Chatbot using GPT for general Q&A and document-based retrieval.
- AI assistant to extract answers from PDFs and notes.
- Country Information Agent: Provides capital, PM, and President of any country.
- Streamlit UI for document-based intelligent QA system.
- Custom trained vector search-based agents using local documents.
